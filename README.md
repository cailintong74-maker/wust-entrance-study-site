# WUST Entrance Exam Study Site

Static study website for Wrocław University of Science and Technology English-taught Bachelor entrance test preparation.

## Local Use

Open:

```text
site/index.html
```

## GitHub Pages

This repository includes a GitHub Actions workflow that publishes the `site/` folder to GitHub Pages after pushes to `main`.

After pushing to GitHub, enable GitHub Pages with **GitHub Actions** as the source in the repository settings.
